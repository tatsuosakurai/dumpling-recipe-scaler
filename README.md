# 蔓餃苑 餃子分量計算

作りたい餃子の個数を入力すると、肉餡・野菜餡・皮の分量を自動計算する静的ウェブアプリです。

## 使い方

`index.html` をブラウザで開くだけで使えます。

## GitHub Pages で公開する

1. このディレクトリを GitHub リポジトリに push する
2. GitHub の `Settings` から `Pages` を開く
3. `Build and deployment` の `Source` で `Deploy from a branch` を選ぶ
4. `Branch` で `main` と `/ (root)` を選んで保存する

数分後に GitHub Pages の URL で公開されます。

公開URL:

https://tatsuosakurai.github.io/dumpling-recipe-scaler/
